# 📐 Controllo Triangolo Rettangolo 🧮

Un pratico script in **Python** 🐍 che permette di verificare se una serie di triangoli sono rettangoli, applicando in modo automatico il celebre **Teorema di Pitagora**.

---

## 🚀 Come Funziona?

Il programma guida l'utente passo dopo passo attraverso la console:
1. 🔢 Chiede inizialmente **quanti triangoli** si desidera analizzare.
2. 📏 Per ogni triangolo, richiede l'inserimento della lunghezza dei **tre lati** (accetta anche numeri decimali!).
3. 🕵️‍♂️ **Trova l'ipotenusa:** Identifica automaticamente il lato maggiore (ipotenusa) e i due lati rimanenti (cateti), indipendentemente dall'ordine in cui sono stati inseriti.
4. 🧪 **Verifica:** Applica la formula matematica del Teorema di Pitagora:
   $$i^2 = c1^2 + c2^2$$
5. 📢 Stampa istantaneamente il verdetto finale!
